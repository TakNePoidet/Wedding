# Приёмник RSVP-заявок

Маленький воркер Cloudflare: принимает ответ гостя с сайта и пересылает
его в Телеграм. Сайт остаётся статикой на GitHub Pages, ничего не переезжает.

**Зачем он вообще нужен.** Чтобы отправить сообщение боту, нужен его токен.
Если положить токен в код сайта, он попадёт в собранные файлы — а они
публичны, токен достанет любой и сможет слать что угодно в ваш чат.
Поэтому токен живёт здесь, в секретах Cloudflare, и наружу не выходит.

## Настройка (один раз)

### 1. Бот и чат

1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot` → получите **токен**
   вида `1234567890:AAF...`.
2. Создайте чат (или группу), куда будут падать заявки, и добавьте туда бота.
3. Узнайте **chat_id**: напишите в этот чат любое сообщение и откройте
   `https://api.telegram.org/bot<ТОКЕН>/getUpdates` — нужное число лежит
   в `result[0].message.chat.id`. У групп оно отрицательное, минус — часть числа.

### 2. Деплой воркера

```bash
cd worker
npx wrangler login            # откроется браузер, войдите в Cloudflare
npx wrangler secret put TELEGRAM_TOKEN     # вставьте токен
npx wrangler secret put TELEGRAM_CHAT_ID   # вставьте chat_id
npx wrangler deploy
```

Последняя команда напечатает адрес вида
`https://wedding-rsvp.ваш-аккаунт.workers.dev`.

### 3. Подключение сайта

Впишите этот адрес в `RSVP_ENDPOINT` в `src/data/wedding.ts` и задеплойте сайт.
Пока строка пустая, форма не отправляет ничего и просит гостя написать
организатору — сломанной она не выглядит.

## Проверка

```bash
curl -X POST https://wedding-rsvp.<ваш>.workers.dev \
  -H 'Content-Type: application/json' \
  -H 'Origin: https://leranikita.ru' \
  -d '{"first":"Тест","attending":true,"drinks":["Виски"]}'
```

Ответ `{"ok":true}` и сообщение в чате — значит всё работает.

## Что стоит знать

- **Домен зашит** в `ALLOWED_ORIGIN` в `wrangler.toml`. Заявки с других
  источников отклоняются. Защита слабая — заголовок `Origin` подделывается
  чем угодно, кроме браузера, — но отсекает ботов, ползающих по всему интернету.
  От целенаправленного спама она не спасёт; если начнётся, самое быстрое —
  сменить адрес воркера (`name` в `wrangler.toml`) и обновить `RSVP_ENDPOINT`.
- **Заявки нигде не хранятся.** Воркер только пересылает; если сообщение
  в Телеграме потерялось, восстановить его неоткуда. Для архива есть
  таблица «Свадьба 13.09.2026 — ответы гостей» на Google Диске —
  переносите туда руками.
- Бесплатного тарифа Cloudflare (100 000 запросов в сутки) хватает
  с запасом в тысячи раз.
