# Свадебный сайт · Никита и Лера

Одностраничное приглашение на свадьбу **13 сентября 2026**, ресторан «Лебединое озеро»
(ПКиО им. М. Гафури, проспект Октября, 77/2, Уфа).

Статический сайт без сборки и зависимостей — только `index.html`, `styles.css`, `script.js`.

## Структура

| Файл | Что внутри |
|------|-----------|
| `index.html` | Разметка всех секций: обложка, обратный отсчёт, детали, карта, дресс-код, RSVP |
| `styles.css` | Дизайн-система (палитра и шрифты — токенами в `:root`), адаптив, анимации |
| `script.js`  | Таймер обратного отсчёта, RSVP-кнопка, плавное появление секций |

## Разделы

- **Обложка** — имена, дата, место
- **Обратный отсчёт** — таймер до 13.09.2026 15:00 (Уфа, UTC+5)
- **Детали дня** — дата, время, место
- **Карта проезда** — встроенная Яндекс.Карта + кнопки Яндекс / 2ГИС / Google
- **Дресс-код** — палитра природных оттенков и рекомендации
- **RSVP** — кнопка подтверждения присутствия через мессенджер

## Что нужно заменить перед публикацией

1. **Контакт для RSVP** — в начале `script.js`, объект `RSVP_CONFIG`:
   ```js
   var RSVP_CONFIG = {
     type: 'whatsapp',        // 'whatsapp' | 'telegram' | 'phone'
     phone: '79991234567',    // номер без + и пробелов
     telegram: '',            // ник без @ (если type: 'telegram')
     message: '...'           // текст, который подставится в сообщение
   };
   ```
   Пока стоит номер-заглушка `70000000000` — под кнопкой видна подсказка о настройке.

2. **Точка на карте** (по желанию) — сейчас метка ставится по адресу. Чтобы задать
   точную точку, откройте [Яндекс.Конструктор карт](https://yandex.ru/map-constructor/),
   получите ссылку вида `.../map-widget/v1/?ll=<lon>,<lat>&z=16&pt=<lon>,<lat>` и вставьте
   её в `src` у `<iframe>` в секции «Карта» в `index.html`.

3. **Срок ответа RSVP** — при необходимости поправьте дату «1 августа» в `index.html`.

## Как посмотреть локально

Просто откройте `index.html` в браузере. Либо поднимите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Публикация на GitHub Pages

1. Запушьте файлы в репозиторий.
2. **Settings → Pages** → *Build and deployment* → *Source: Deploy from a branch*.
3. Выберите ветку и папку `/ (root)`, сохраните.
4. Через минуту сайт будет доступен по адресу `https://<владелец>.github.io/<репозиторий>/`.

поддерживают кириллицу). При недоступности сети подставляются системные serif/sans —
вёрстка не ломается.
