# assets

Сюда кладём картинки сайта.

## couple.png — Funko-персонажи для обложки
- Формат: **PNG с прозрачным фоном** (без белой/серой подложки).
- Пара вместе (жених + невеста), как на референсе.
- Ширина ~1000–1400 px, вертикальная/квадратная ориентация.
- Файл должен называться ровно `couple.png` и лежать в `assets/`.

Пока файла нет — блок с персонажами на обложке автоматически скрывается,
вёрстка остаётся прежней (текст + ботаника). Как добавите файл — персонажи
появятся справа от имён (на мобильном — под ними).

Можно также прислать PNG мне — добавлю в репозиторий сам.
