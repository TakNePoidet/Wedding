# assets

Картинки сайта. Все — Funko-персонажи на **прозрачном фоне** (PNG).

| Файл | Где на сайте |
|------|--------------|
| `couple.png` | Обложка — пара стоят рядом (правый низ на десктопе, под именами на мобиле) |
| `groom-wave.png` | «Приглашение» — жених машет рукой |
| `bride-bouquet.png` | «О нарядах» — невеста с букетом зелени |

Персонажи подключены декоративно (`aria-hidden`), с лёгким «дыханием»
(отключается при `prefers-reduced-motion`). Если файл убрать — блок с
персонажем аккуратно исчезает, вёрстка не ломается.

Чтобы заменить персонажа — положите новый PNG с тем же именем (прозрачный фон,
вертикальная/квадратная ориентация, ~800–1200 px).
